# ANSYS Simulation Data

The complete ANSYS simulation files (including project files, animations, and contour outputs) are hosted externally due to size constraints.

Access is provided with respect to the following links mentioned below:

* Anisotropic simulations:
  https://drive.google.com/drive/folders/1f6OJ_OwyqfrxTYT6ATp0n3RXzPYR7--k?usp=sharing

* Berkovich indentation simulations:
  https://drive.google.com/drive/folders/1jdm1pDnhkzElL95ii5O7oCCXW3uzElkR?usp=sharing

* Vickers indentation simulations:
  https://drive.google.com/drive/folders/14GfCwlmdP4O3ISKLyaRsSJWZ9ghPExvi?usp=sharing

The repository contains all necessary scripts and input files to reproduce these simulations.
